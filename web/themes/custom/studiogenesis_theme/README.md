# Studiogenesis Theme

Tema personalizado del proyecto Drupal para Studiogenesis.

## Objetivo

- Usar Bootstrap 4 como base visual local.
- Mantener una capa propia de estilos sencilla y profesional.
- Dar soporte a las paginas publicas de noticias, productos y newsletter.

## Bootstrap

Bootstrap 4.6.2 esta incluido localmente en `vendor/bootstrap` para que la entrega no dependa de un CDN externo.

## Identidad visual

- El logo principal del tema es `logo.jpg`.
- La paleta se basa en el azul del logo de Studiogenesis.
- Los estilos propios viven en `css/styles.css`.
- La plantilla principal del sitio vive en `templates/layout/page.html.twig`.
